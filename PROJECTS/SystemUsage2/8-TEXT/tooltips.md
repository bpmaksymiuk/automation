# Tooltip & Help Copy — The Boiler Room

## Tooltip patterns (exact value on hover/tap)
- CPU gauge: CPU {value}% - alarm at {threshold}%
- GPU gauge: GPU {value}% - VRAM {usedMB}/{totalMB} MB
- Memory tank: Memory {value}% - {usedGB}/{totalGB} GB - swap {swapPercent}%
- Disk tank: {mount} {value}% - {usedGB}/{totalGB} GB
- Network pipe: up {tx} / down {rx}

## History modal
- Title pattern: {RESOURCE} - last 30 minutes
- Close button: Close
- Empty state: No data yet - let the boiler run a moment.

## First-run hint
- One line: Hover any gauge for exact figures. Drag a valve in Alert Control to set your limits.
