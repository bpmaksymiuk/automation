# UI Strings — The Boiler Room

## Header
- App title: THE BOILER ROOM
- Connection states: live | demo | disconnected

## Panel titles (tactile, metaphor-led)
- CPU panel: CPU FURNACE
- GPU panel: GPU FORGE
- Memory panel: MEMORY BOILER
- Disk panel: DISK PRESSURE TANKS
- Network panel: NETWORK STEAM PIPES
- Alert panel: ALERT CONTROL

## Readout labels
- CPU: Overall / Per-core / Temp
- GPU: Utilization / VRAM
- Memory: Used / Available / Swap
- Disk: read / write
- Network: up (transmit) / down (receive)

## Buttons
- Reduced motion toggle (off): Calm the room
- Reduced motion toggle (on): Wake the room
- Alert drawer toggle: Alert Control
- Demo load button: Stoke the furnace

## Process tables
- Column headers: Process / PID / Load

## Empty / unavailable states
- GPU absent: GPU not available
- History empty: No data yet - let the boiler run a moment.
