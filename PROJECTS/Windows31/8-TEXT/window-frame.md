# Window Frame Copy Contract

## WindowFrame

| Field key | Character limit | Final copy | Notes |
|---|---:|---|---|
| `window_close_button_label` | 10 | `Close` | Shared control label for the common close action. |

## Tone Application

- Accessibility labels should describe what the control is, not narrate hidden behavior.
- Shared chrome language stays literal and stable across all windows.
