# Boot Sequence Copy Contract

## BootSequence

| Field key | Character limit | Final copy | Notes |
|---|---:|---|---|
| `boot_loading_label` | 28 | `Starting Windows 3.1...` | Ceremonial startup line with instant historical context. |
| `boot_status_line` | 48 | `Please wait while Program Manager wakes up.` | Warm secondary line that keeps the shell metaphor intact. |
| `boot_skip_button_label` | 12 | `Skip Boot` | Direct action label visible from first paint. |

## Tone Application

- Keep the boot ritual affectionate, not ironic.
- Favor short lines that still read cleanly during motion or on small screens.
