# Secret Overlay Copy Contract

## SecretOverlay

| Field key | Character limit | Final copy | Notes |
|---|---:|---|---|
| `secret_window_title` | 24 | `After Hours` | Distinct but still grounded in the desktop world. |
| `secret_headline` | 40 | `You found the after-hours room.` | Small reward line that feels discovered rather than announced. |
| `secret_body` | 180 | `The office lights are low, the icons are still awake, and Program Manager is letting you peek behind the curtain. Close this window to drift back to the desktop.` | Secret copy stays magical, warm, and clearly dismissible. |
| `secret_dismiss_button_label` | 20 | `Back to Desktop` | Clear return path that restores the shell. |

## Tone Application

- Let the secret feel special without breaking the historical frame.
- Keep dismissal language explicit so the reward never becomes a trap.
